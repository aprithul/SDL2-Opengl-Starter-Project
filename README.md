# SDL2-Opengl-Starter-Project
Makefile based starter project for SDL2 + OpenGL. 


# Easy Setup

Pre-requisites:
  Homebrew and Clang++

Install:
* Open terminal in folder and chane execution permission of setup.sh by running:
  chmod 755 setup.sh
* execute setup.sh:
  ./setup.sh
* open workspace and run project (fn + f5)
